I am interested in programming.

You might have already guessed it by the fact that I have created an entire website, but after all, we're all just learning.

You're right. HTML is not a programming language. But I don't care.

My favorite languages.

I like to use the languages which are suitable for the jobs I need to do. So for example:

  • Dynamic webpage content - Javascript
  • Backend programming - Javascript (Node.js)
  • Static site generation - Javascript (React)
  • Mobile apps - Javascript (React Native)
  • Machine learning - Javascript (Tensorflow.js)
  • Desktop apps - Javascript (Electron framework)
  • 3D games - Javascript (polygon.js)

The worst part is that those are actually true.

What is the future of programming?

Contrary to popular belief, I don't think AI will replace programmers. Mathematicians somehow survived the invention of the calculator, so there is nothing to worry about.

We should be more worried about the rise of content management systems, like Wordpress. We are being flooded by webpages created by people who know nothing about programming, but have pressed some buttons, dragged some boxes, and with the magic of a user-friendly CMS - they have a webpage!
